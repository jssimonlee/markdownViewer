# MD Viewer

순수 HTML/CSS/JS로 만든 **로컬 마크다운 뷰어**입니다.  
별도 서버 없이 `index.html`을 더블클릭하면 바로 실행됩니다.

---

## 주요 기능

### 📂 파일 열기
- 툴바 **열기** 버튼 클릭
- 파일을 뷰어 창에 **드래그 앤 드롭**
- 최근 열었던 파일을 **사이드바 최근 파일** 목록에서 바로 열기

### 👁️ 마크다운 렌더링
- `marked.js`를 이용한 마크다운 파싱 (GFM 지원)
- `highlight.js`를 이용한 코드 블록 **문법 강조**
- 코드 블록 우상단 **복사 버튼** 제공
- 표(Table), 체크리스트, 인용문, 이미지 등 전체 문법 지원

### 📝 편집 모드 (Split View)
- 툴바 **편집** 버튼 또는 `Ctrl+E` 단축키로 진입
- 좌측 에디터 ↔ 우측 실시간 미리보기 **분할 화면**
- **새 파일** 버튼으로 빈 파일을 편집 모드로 시작
- 편집 중 다른 파일 열기·새 파일·최근 파일 클릭 시 **저장 여부 경고** 표시

### 💾 저장
- 툴바 **저장** 버튼 또는 `Ctrl+S` (브라우저 기본 동작과 별개)
- Chrome/Edge: **저장 위치 선택 다이얼로그** (`showSaveFilePicker`)
- 그 외 브라우저: 기본 다운로드 폴더로 자동 저장 (폴백)

### 📑 목차 (TOC)
- 마크다운의 `h1` ~ `h3` 헤딩을 자동으로 파싱해 **사이드바에 목차** 생성
- 스크롤 위치에 따라 현재 섹션 **하이라이트** (IntersectionObserver)

### 📖 MD 가이드
- 사이드바 하단 **MD 가이드** 패널 (클릭해서 펼치기)
- 제목·텍스트 서식·목록·링크·코드·표·기타 7개 카테고리
- **편집 모드**: 항목 클릭 시 커서 위치에 문법 **바로 삽입**
- **뷰어 모드**: 항목 클릭 시 클립보드에 **복사**

### 🌓 다크 / 라이트 모드
- 툴바 아이콘 버튼(☀️/🌙)으로 테마 전환 또는 `Ctrl+Shift+T`
- 선택한 테마는 `localStorage`에 저장되어 다음 실행 시에도 유지

---

## 단축키

| 단축키 | 동작 |
|---|---|
| `Ctrl + E` | 편집 / 미리보기 모드 전환 |
| `Ctrl + Shift + T` | 다크 / 라이트 테마 전환 |
| `Ctrl + \` | 사이드바 열기 / 닫기 |

---

## 기술 스택

| 항목 | 내용 |
|---|---|
| 마크다운 파싱 | [marked.js](https://marked.js.org/) v4.3.0 |
| 문법 강조 | [highlight.js](https://highlightjs.org/) v11.9.0 |
| 폰트 | Google Fonts — Inter, Fira Code |
| 저장 API | File System Access API + Blob 폴백 |
| 빌드 도구 | 없음 (순수 HTML/CSS/JS) |

## 브라우저 호환성

| 기능 | Chrome | Edge | Firefox |
|---|---|---|---|
| 기본 렌더링 | ✅ | ✅ | ✅ |
| 저장 위치 선택 | ✅ | ✅ | ⬛ (자동 다운로드) |
